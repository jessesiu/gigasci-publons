   Yii framework  //website framework 
   
   PHP            //website coding language
   
   JavaScript     //website form 
   
   Publon API     //https://publons.com/api/v1/
   
   DataCite API   //https://mds.datacite.org/static/apidoc
   
   Postgresql     //database
